# Node.js Basic
This repository contains basic examples and tutorials for Node.js, a powerful JavaScript runtime built on Chrome's V8 engine. Whether you're new to Node.js or looking to refresh your knowledge, this collection of resources will help you get started with building server-side applications using Node.js.
## Getting Started
To get started with Node.js, you need to have it installed on your machine. You can
download it from the official website: [https://nodejs.org/](https://nodejs.org/). Once you have Node.js installed, you can create a new project and start writing your first Node.js application.
```bash
mkdir node-js-basic
cd node-js-basic
npm init -y
```	
This will create a new directory for your project and initialize it with a `package.json` file. You can then create a new JavaScript file, for example `app.js`, and start writing your Node.js code.
```javascript
// app.js
console.log('Hello, Node.js!');
```
To run your Node.js application, use the following command in your terminal:
```bash
node app.js
```	
This will execute your `app.js` file and print "Hello, Node.js!" to the console. From here, you can explore the various features and capabilities of Node.js to build more complex applications.
